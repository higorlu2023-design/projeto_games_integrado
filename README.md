# projeto_games_integrado

Contexto do projeto

Neste projeto, realizei uma análise de dados para a loja online Ice, que comercializa videogames em diferentes regiões do mundo.
Utilizando dados históricos de vendas de jogos até 2016, avaliações de usuários e críticos, gêneros e plataformas — como Xbox, PlayStation e PC —, busquei identificar padrões que ajudam a explicar o sucesso comercial de um jogo.
A análise foi construída considerando o cenário de dezembro de 2016, com o objetivo de apoiar o planejamento de campanhas publicitárias para 2017. A partir dos resultados, foi possível identificar plataformas, gêneros e regiões com maior potencial de vendas, contribuindo para decisões de marketing mais orientadas por dados.


Perguntas para análise

- Para orientar a análise, foram definidas as seguintes perguntas:
Quantos jogos foram lançados a cada ano e quais anos possuem dados suficientes para uma análise confiável?

- Como as vendas variaram entre as plataformas ao longo do tempo?

- Quais plataformas tiveram maior volume de vendas, quais cresceram, quais entraram em declínio e qual é o ciclo de vida aproximado de uma plataforma?

- Qual período de dados é mais relevante para planejar uma campanha para 2017?

- Quais plataformas apresentaram maior potencial de vendas no período selecionado?

- Como as vendas globais dos jogos se distribuem entre as plataformas? Existem diferenças na mediana, na dispersão e na presença de jogos com vendas muito altas?

- As avaliações dos críticos possuem relação com as vendas globais dos jogos em uma plataforma popular?

- Como os mesmos jogos se comportam em plataformas diferentes?

- Quais gêneros apresentaram maior desempenho em vendas nas plataformas selecionadas?

- Como os jogos mais vendidos se comportaram em cada região: América do Norte, Europa, Japão e outros mercados?

- Quais regiões e plataformas devem ser priorizadas em uma possível campanha para uma continuação de Grand Theft Auto V?
